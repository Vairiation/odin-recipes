# odin-recipes
Project: Recipes - HTML section of the Foundations course on the Odin Project

This project will be used to practice and work on the learnings of the HTML section of the Foundations course in the Odin Project

Things covered include:

- Elements
- Attributes
- Lists (ordered and unordered)
- Images
- Links
- Headings
- Subheadings
- Head
- Body
- Titles
- Meta Data
- Etc. (more to be added later if needed)
